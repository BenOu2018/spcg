# SPCG 关卡内晋级示意图

本目录用于关卡内题目进度与晋级引导。

- `promote-page-review-v1.png`：AI 生成的视觉设计稿。
- `promote-page-template.html`：可直接打开预览的前端模板。
- `promote.css`：模板样式。
- `*.svg`：节点、徽章、按钮和图标组件。
- `promote-titlebar-review-v2.png`：标题栏紧凑版设计稿。
- `promote-titlebar-template.html`：标题栏紧凑版前端模板。
- `promote-titlebar.css`：标题栏紧凑版样式。

## 交互规则

- 1-3 题是必做题，全部完成后才允许晋级下一关。
- 4-5 题是提高/挑战题，不阻塞晋级。
- 当前题目使用高亮发光状态。
- 必做题完成后仍引导学生尝试提高题，用于获得额外称谓、金币或掌握反馈。
- 标题栏紧凑版建议高度控制在 `104px-132px`，圆形徽章可居中轻微下探，不影响 IDE 主编辑区。
