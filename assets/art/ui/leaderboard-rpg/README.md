# SPCG RPG Leaderboard

This folder contains the RPG adventure leaderboard design.

- `leaderboard-rpg-mockup-v1.png`: imagegen reference mockup.
- `leaderboard-rpg.html`: standalone frontend design page.
- `leaderboard-rpg.css`: standalone CSS for the design page.
- `svg/`: deterministic SVG UI assets used by both the design page and `/leaderboard`.

Direction: open-world RPG adventure guild ranking board, adapted for SPCG programming leaderboard data.
