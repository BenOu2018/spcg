两个袋子的秘密

小镇的仓库里有两个袋子，分别装着不同数量的星星石。请写一段程序，读取两个整数， 输出交换后的结果。

输入

一行两个整数 a 和 b。

输出

交换后的 b 和 a。

样例 输入: 3 8
输出: 8 3
提示：可以用一个临时变量，也可以思考更简洁的写法。

#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    int temp = a; // 保存第一个袋子
    a = b;
    b = temp;
    cout << a << " " << b;
}
