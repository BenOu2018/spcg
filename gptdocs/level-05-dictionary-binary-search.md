# 第 5 关：字典里找字

## 题目背景

小狗在镜湖小径旁发现了一本旧字典。字典里的词条已经按照字母顺序排好，每个词条对应一张通往下一站的提示卡。

现在，小狗想快速找到老师给出的那个词条。一本字典很厚，如果从第一页慢慢翻会很慢。请你帮助小狗使用“每次从中间找”的方法，判断目标词条在字典中的位置。

## 题目描述

给定 `n` 个已经按字典序从小到大排列的单词，再给定一个目标单词 `target`。

如果 `target` 出现在字典中，输出它第一次出现的位置编号；位置编号从 `1` 开始。

如果没有找到，输出 `-1`。

## 输入格式

第一行输入一个整数 `n`，表示字典中的单词数量。

第二行输入 `n` 个按字典序从小到大排列的单词。

第三行输入一个目标单词 `target`。

## 输出格式

输出一个整数：

- 如果找到目标单词，输出它的位置编号。
- 如果没有找到，输出 `-1`。

## 数据范围

- `1 <= n <= 100000`
- 单词只包含小写英文字母
- 每个单词长度不超过 `20`
- 输入保证第二行单词按字典序从小到大排列
- 如果有重复单词，需要输出第一次出现的位置

## 输入样例 1

```text
7
apple book cloud dog moon river star
moon
```

## 输出样例 1

```text
5
```

## 样例解释 1

`moon` 是字典中的第 `5` 个单词，所以输出 `5`。

## 输入样例 2

```text
6
ant cat door fish kite tree
bird
```

## 输出样例 2

```text
-1
```

## 样例解释 2

字典中没有 `bird`，所以输出 `-1`。

## 提示

字典已经排好序，可以每次查看中间位置的单词：

- 如果中间单词正好是目标，就继续向左确认是否还有更早的位置。
- 如果中间单词比目标小，目标只可能在右半边。
- 如果中间单词比目标大，目标只可能在左半边。

这就是二分查找。
