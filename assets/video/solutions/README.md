# Solution Videos

系统题解视频统一放在这里，数据库只保存链接。

推荐路径：

```text
assets/video/solutions/{chapterId}/{levelId}.mp4
```

数据库字段：

```text
solution_video_url
```

前端/内容模板中使用的链接：

```text
/video/solutions/{chapterId}/{levelId}.mp4
```

例：

```text
assets/video/solutions/ch1-mist-town/ch1-01.mp4
/video/solutions/ch1-mist-town/ch1-01.mp4
```
