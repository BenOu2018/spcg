Chapter 1: Mist Town
Task

Collect all glowing orbs

Goal

Write a program that guides the character through the map and collects every glowing orb.

Rules

  • You can move up, down, left, or right.
  • You cannot walk through rocks.
  • Collect all orbs before reaching the exit.

Input

The map size and tile information are provided by the judge.

Output

Print one valid route command sequence.

Example

move("right") move("down") move("right")
C++ Editor 
// Use move() to reach all orbs.
// Available: "up", "down", "left", "right"

int main() {
    move("right");
    move("right");
    move("down");
    move("left");
    move("up");
    return 0;
}
