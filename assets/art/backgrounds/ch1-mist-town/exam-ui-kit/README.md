# SPCG 段位赛界面 UI Kit

## 产物

- `exam-page-review-v1.png`：段位赛编程页视觉稿。
- `exam-page-template.html`：可直接打开的前端页面模版。
- `*.svg`：木质顶栏、SPCG 牌匾、段位赛标题、题号/计时、视频监控、摄像头预览、面板四角、段位赛操作按钮和 IDE 小图标等组件。
- `*.png`：由 SVG 渲染出的备用位图资源。

## 设计边界

- 题目栏和 IDE 主结构沿用编程关卡页面。
- 段位赛只新增顶部段位赛身份、题号/计时、视频监控状态和结束段位赛操作。
- 视频监控必须紧凑，不占用题目栏和 IDE 空间。
- 监控状态表达为“Camera On / Recording / secure preview”，避免沉重的监控压迫感。

## 还原重点

- 顶部段位赛栏采用暗木横梁和暗金描边，题号、计时、监控是独立牌匾。
- 左侧题目栏采用羊皮纸、细内框和四角装饰。
- 右侧 IDE 不使用 tab 标签，代码画布占主要空间，工具图标收在右上角。
- 底部结果 dock 保留 `运行结果 / 测试用例 / 评测输出`，主操作固定在右下角。
